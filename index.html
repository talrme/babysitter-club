Firebase setup checklist

  1. Create a Firebase project and add a Web app.
  2. Copy the config object into firebase-config.js.
  3. Enable Authentication → Sign-in method → Google.
  4. Add authorized domains: localhost, 127.0.0.1, and talrme.github.io.
  5. Run this app with a local server (not file://), then click sign in.

Live site: https://talrme.github.io/babysitter-club. Detailed instructions are in AI_README.md.
